IOT

The internet of things, or IoT, is a system of interrelated computing devices, mechanical and digital machines, objects, animals or people that are provided with unique identifiers (UIDs) and the ability to transfer data over a network without requiring human-to-human or human-to-computer interaction.

IoT BASED GREEN ENERGY MANAGEMENT

This Project focuses on producing electricity through mechanical effects that are from revolving door and speed breakers. This includes chain drive mechanism,gear base mechanism to generate power. The generated power is stored in the battery and used for various purposes later. Further Stored power is monitored through IoT. In the Proposed Project, Current sensor is a device that detects electric Current and display it using IoT. Later, the data is analyzed and can be used for the purpose of control.


Outcome of this project allows the user to control the selected loads wireless and allows to decide which crucial loads to run in the event of power failure through a GUI such as a Web page/Mobile Application. To then display to the user as to how long he can run the chosen loads simultaneously or individually.
